**data set & dataset**
dataset 和 data set 的含义有微妙的区别。data 和 set 本来是两个词，一般来说用 the data set(s) 是不会错的，也很正规，注意一定要加冠词，来指出特定的数据集，dataset 作为一个新词相对于 data set 的抽象层次提升了，用来指具有数据集属性的东西，比如网上各种公开数据集，可以叫public datasets。写论文时用 data set 比较保险。
